.. _glossary:

********
Glossary
********

.. glossary::

   API
      Application program interface.

   solution
      An object of type :class:`exactpack.base.ExactSolution`, which
      represents the solution produced by a particular :term:`solver`.

   solver
      An object of a type derived from
      :class:`exactpack.base.ExactSolver`.  This is a function-like
      object which, when called, will return a :term:`solution` object
      for the specified problem.
